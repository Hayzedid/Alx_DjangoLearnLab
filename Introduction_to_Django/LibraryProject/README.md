# LibraryProject

A Django project for learning Django development fundamentals.

## Setup

1. Install Django: `pip install django`
2. Navigate to project directory: `cd LibraryProject`
3. Run development server: `python manage.py runserver`
4. Open browser to: http://127.0.0.1:8000/

## Project Structure

- `settings.py`: Configuration for the Django project
- `urls.py`: URL declarations for the project; a "table of contents" of your Django-powered site
- `manage.py`: Command-line utility that lets you interact with this Django project

## Apps

- `bookshelf`: App for managing books with Book model
